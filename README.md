# Auto Clicker script for Roblox
### by SkLp

## Описание
Автокликер для Roblox с кастомным курсором и интерфейсом.

## Возможности
- Автоматический кликер с настраиваемой скоростью клика
- Кастомный курсор
- Интерфейс с переключением языка (RU/EN)
- Горячие клавиши: F6, F5, F4
