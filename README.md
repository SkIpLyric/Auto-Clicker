# Auto Clicker for Roblox
### by SkLp

## Описание
Автокликер для Roblox с кастомным курсором и интерфейсом.

## Возможности
- Автоматический кликер с настраиваемой скоростью
- Кастомный курсор в стиле Mac
- Интерфейс с переключением языка (RU/EN)
- Горячие клавиши: F6, F5, F4
